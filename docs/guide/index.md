4. **Core concepts** — [Namespaces](/guide/concepts/namespaces),

   [Scopes](/guide/concepts/scopes),

5. [Form metadata](/guide/form-metadata) and

6. [Migration](/guide/migration) — coming from a pre-namespace version.
7. [FAQ](/guide/faq).
