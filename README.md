# typespec-libraries

[![CI](https://github.com/qninhdt/typespec-libraries/actions/workflows/ci.yml/badge.svg)](https://github.com/qninhdt/typespec-libraries/actions/workflows/ci.yml)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

**Define your database schema once in TypeSpec. Generate production-ready Go (GORM) and Python (SQLModel) models automatically.**

Built on [TypeSpec](https://typespec.io) - Microsoft's API specification language - these libraries bring a single source of truth to your ORM layer. Add decorators to your TypeSpec models and get fully-typed, annotated Go structs and Python classes with zero boilerplate.

## Quick Start

```bash
pnpm add -D @qninhdt/typespec-orm @qninhdt/typespec-gorm @qninhdt/typespec-sqlmodel
```

**`tspconfig.yaml`:**

```yaml
emit:
  - "@qninhdt/typespec-gorm"
  - "@qninhdt/typespec-sqlmodel"
options:
  "@qninhdt/typespec-gorm":
    package-name: "models"
  "@qninhdt/typespec-sqlmodel":
    module-name: "models"
```

## Example

### Input - TypeSpec schema

```typescript
import "@qninhdt/typespec-orm";
using Qninhdt.Orm;

enum PostStatus {
  draft: "draft",
  published: "published",
  archived: "archived",
}

/** A blog post authored by a user. */
@table("posts")
@compositeIndex("idx_posts_author_status", "author_id", "status")
model Post {
  @id id: uuid;

  @maxLength(255)
  title: string;

  body: text;

  @index
  status: PostStatus;

  /** Stored as NUMERIC(10,2) to avoid floating-point drift */
  @precision(10, 2) @map("view_count")
  viewCount: decimal;

  @autoCreateTime @map("created_at") createdAt: utcDateTime;
  @autoUpdateTime @map("updated_at") updatedAt?: utcDateTime;
  @softDelete     @map("deleted_at") deletedAt?: utcDateTime;

  @foreignKey("users", "id") @onDelete("CASCADE")
  author: User;
}
```

### Output - Go (GORM)

```go
// Code generated by @qninhdt/typespec-gorm. DO NOT EDIT.
// Source: https://github.com/qninhdt/typespec-libraries

package models

import (
    "time"

    "github.com/google/uuid"
    "github.com/shopspring/decimal"
    "gorm.io/gorm"
)

// PostStatus represents the post_status enum.
type PostStatus string

const (
    PostStatusDraft     PostStatus = "draft"
    PostStatusPublished PostStatus = "published"
    PostStatusArchived  PostStatus = "archived"
)

// Post A blog post authored by a user.
type Post struct {
    ID        uuid.UUID       `gorm:"column:id;type:uuid;primaryKey;default:gen_random_uuid()" json:"id"`
    Title     string          `gorm:"column:title;type:varchar(255);not null" validate:"required,max=255" json:"title"`
    Body      string          `gorm:"column:body;type:text;not null" validate:"required" json:"body"`
    Status    PostStatus      `gorm:"column:status;type:varchar(9);not null;index;index:idx_posts_author_status,priority:2" validate:"required,oneof=draft published archived" json:"status"`
    ViewCount decimal.Decimal `gorm:"column:view_count;type:numeric(10,2);not null" json:"viewCount"`
    CreatedAt time.Time       `gorm:"column:created_at;type:timestamptz;not null;autoCreateTime" json:"createdAt"`
    UpdatedAt *time.Time      `gorm:"column:updated_at;type:timestamptz;autoUpdateTime" json:"updatedAt,omitempty"`
    DeletedAt gorm.DeletedAt  `gorm:"column:deleted_at;index" json:"deletedAt"`
    AuthorID  uuid.UUID       `gorm:"column:author_id;type:uuid;not null;index;index:idx_posts_author_status,priority:1;constraint:OnDelete:CASCADE" json:"authorId"`

    // в”Ђв”Ђв”Ђ Relationships в”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђ

}

// TableName returns the table name for Post.
func (Post) TableName() string { return "posts" }
```

### Output - Python (SQLModel)

```python
# Code generated by @qninhdt/typespec-sqlmodel. DO NOT EDIT.
# Source: https://github.com/qninhdt/typespec-libraries

from datetime import datetime
from decimal import Decimal
from enum import Enum
from uuid import UUID, uuid4

from sqlalchemy import Column, DateTime, ForeignKey, Index, Numeric, func
from sqlalchemy import Enum as SAEnum
from sqlmodel import Field, Relationship, SQLModel


class PostStatus(str, Enum):
    """Auto-generated enum for post_status."""

    draft = "draft"
    published = "published"
    archived = "archived"


class Post(SQLModel, table=True):
    """A blog post authored by a user."""

    __tablename__ = "posts"
    __table_args__ = (
        Index("idx_posts_author_status", "author_id", "status"),
    )

    id: UUID = Field(default_factory=uuid4, primary_key=True)
    title: str = Field(max_length=255, sa_column_kwargs={"nullable": False})
    body: str = Field(sa_column_kwargs={"nullable": False})
    status: PostStatus = Field(
        sa_column=Column(SAEnum(PostStatus), nullable=False, index=True))
    view_count: Decimal = Field(sa_column=Column(Numeric(10, 2), nullable=False))
    created_at: datetime = Field(
        sa_column=Column(DateTime(timezone=True), nullable=False, server_default=func.now()))
    updated_at: datetime | None = Field(
        default=None,
        sa_column=Column(DateTime(timezone=True), onupdate=func.now(), server_default=func.now()))
    deleted_at: datetime | None = Field(
        default=None, sa_column=Column(DateTime(timezone=True), index=True))
    author_id: UUID = Field(
        sa_column=Column(ForeignKey("users.id", ondelete="CASCADE"), nullable=False))

    # в”Ђв”Ђв”Ђ Relationships в”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђв”Ђ
    author: "User" | None = Relationship(back_populates="posts")
```

## Type Mapping Reference

> All types target **PostgreSQL**. Apply `@precision(p, s)` on `decimal`/`float64` to override the column type.

| TypeSpec type | PostgreSQL         | Go (GORM)           | Python (SQLModel)         |
| ------------- | ------------------ | ------------------- | ------------------------- |
| `uuid`        | `uuid`             | `uuid.UUID` В№       | _(inferred)_ ВІ            |
| `string`      | `varchar(255)`     | `string`            | `String(255)`             |
| `text`        | `text`             | `string`            | `Text`                    |
| `boolean`     | `boolean`          | `bool`              | `Boolean`                 |
| `int8`        | `smallint`         | `int8`              | `SmallInteger`            |
| `int16`       | `smallint`         | `int16`             | `SmallInteger`            |
| `int32`       | `integer`          | `int32`             | `Integer`                 |
| `int64`       | `bigint`           | `int64`             | `BigInteger`              |
| `uint8`       | `smallint`         | `uint8`             | `SmallInteger`            |
| `uint16`      | `integer`          | `uint16`            | `Integer`                 |
| `uint32`      | `bigint`           | `uint32`            | `BigInteger`              |
| `uint64`      | `bigint`           | `uint64`            | `BigInteger`              |
| `float32`     | `real`             | `float32`           | `Float`                   |
| `float64`     | `double precision` | `float64`           | `Double`                  |
| `decimal`     | `numeric`          | `decimal.Decimal` Ві | `Numeric` вЃґ               |
| `serial`      | `serial`           | `int32`             | `Integer`                 |
| `bigserial`   | `bigserial`        | `int64`             | `BigInteger`              |
| `utcDateTime` | `timestamptz`      | `time.Time`         | `DateTime(timezone=True)` |
| `plainDate`   | `date`             | `time.Time`         | `Date`                    |
| `plainTime`   | `time`             | `time.Time`         | `Time`                    |
| `duration`    | `interval`         | `time.Duration`     | `Interval`                |
| `bytes`       | `bytea`            | `[]byte`            | `LargeBinary`             |
| `jsonb`       | `jsonb`            | `datatypes.JSON` вЃµ  | `JSONB` вЃ¶                 |

> В№ `github.com/google/uuid` В· ВІ `from uuid import UUID` (SQLModel infers the column type) В· Ві `github.com/shopspring/decimal` В· вЃґ `from decimal import Decimal` + `sqlalchemy.Numeric` В· вЃµ `gorm.io/datatypes` В· вЃ¶ `sqlalchemy.dialects.postgresql.JSONB`

**`@format` overrides:**

| `@format(...)`    | Go `validate` tag     | Python type         |
| ----------------- | --------------------- | ------------------- |
| `"email"`         | `validate:"email"`    | `pydantic.EmailStr` |
| `"url"` / `"uri"` | `validate:"url"`      | `pydantic.AnyUrl`   |
| `"uuid"`          | `validate:"uuid"`     | -                   |
| `"ipv4"`          | `validate:"ipv4"`     | -                   |
| `"ipv6"`          | `validate:"ipv6"`     | -                   |
| `"hostname"`      | `validate:"hostname"` | -                   |

## Release

This monorepo uses [Changesets](https://github.com/changesets/changesets):

```bash
pnpm changeset          # describe changes interactively
pnpm version-packages   # bump versions + update CHANGELOGs
pnpm release            # build + publish all packages to npm
```

The [Release workflow](.github/workflows/release.yml) runs automatically on every push to `main`.
It opens a "Version Packages" PR via Changesets; merging that PR publishes to npm.

> **Required secret:** add `NPM_TOKEN` to your repository secrets.
